Ceres?

Ceres is a website dedicated to spreading and promoting awareness to the current wildfires in Australia. It features an interactive map, news feed, and additional information for people to not only learn more about the sitation, but also about how they can help.

Map?

Like our slogan, we believe that if people actually see the fires and learn more about them, then they will be equipped with the knowledge to actually fight the fire. The map is created from Folium, a Python Library and the fires are current and up-to-date, webscraped from NASA's dataframes.
